Project Title: DSA Explorer and Visualiser App
1. Project Overview
Develop an interactive Data Structures and Algorithms Explorer and Visualiser that
combines multiple algorithm visualizations with puzzle-like challenges. The app allows users
to:
• Explore data structures and algorithms visually
• Interact with puzzles/games requiring application of learned algorithms and data
structures
• See live animations of stacks, queues, linked lists, trees, graphs, heaps, sorting,
and dynamic programming.
• Use Pygame package for visualisation
